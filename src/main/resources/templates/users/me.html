MetaMask 지갑 연결

판매·구매 전에 한 번만 연결하면 됩니다. Chrome/Brave + MetaMask 확장 프로그램을 쓰세요.

  1. MetaMask를 설치하고 Sepolia 네트워크로 맞춥니다.
  2. 아래 버튼을 누르면 MetaMask 권한 창이 뜹니다. Account 2도 체크한 뒤 연결하세요.
  3. 계정이 여러 개면 사이트에서 연결할 주소를 한 번 더 고릅니다.
  4. 서명 요청을 승인합니다. (ETH는 안 나가고 주소 소유 증명만 합니다.)
  5. 위에 지갑 주소가 보이면 완료 → 상품 등록/결제.

아직 연결 안 됨 → 먼저 이 버튼을 누르세요.
